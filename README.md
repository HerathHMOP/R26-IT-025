# An Intelligent Adaptive Learning Ecosystem for Nursery and Primary Education

## Overview

An AI-powered adaptive learning ecosystem designed for Nursery and Grade 1–5 students. The platform integrates adaptive English learning, Sinhala literacy development, bilingual Mathematics learning, and creativity-focused educational experiences within a personalized and child-friendly digital environment.

---

# Project Components

## Smart and Adaptive Bilingual Mathematics Learning System

### Description

A gamified bilingual Mathematics learning platform supporting adaptive activity generation, behavioral analysis, multilingual learning, and emotion-aware educational experiences.

### Current Progress

* Interactive mathematics activity framework designed
* Bilingual Sinhala-English learning support planned
* Adaptive learning path concepts developed
* Initial learner-level identification process implemented
* Dashboard and reporting structures designed
* AI-based engagement and behavioral analysis research completed
* Initial UI prototypes for gamified mathematics activities created

## Adaptive Learning and Phonetic Scaffolding System (ALPS)

### Description
